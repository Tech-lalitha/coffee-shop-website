# ☕ Coffee Shop Website

A simple and responsive coffee shop website built using HTML and CSS.

## 📌 Project Description

This project is a basic front-end website that showcases a coffee shop layout with a navigation bar, background image, and styled content. It is designed for practice and beginner-level web development.

## 🚀 Features

* Responsive design (basic)
* Navigation bar
* Background image section
* Stylish text and layout
* Clean UI

## 🛠️ Technologies Used

* HTML5
* CSS
* Bootstrap (CDN)
* Font Awesome (icons)

## 📂 Project Structure

```
coffee-shop-website/
│── coffee-exp.html
│── README.md
```

## ▶️ How to Run

1. Download or clone this repository
2. Open the project folder
3. Double-click on `coffee-exp.html`
   **OR** open with Live Server in VS Code
